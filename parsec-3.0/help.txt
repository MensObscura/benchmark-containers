You need to give arguments to execute installed benchmarks:
		docker run [repository/tag] [ACTION] [PACKAGE] [BUILDCONF] [INPUT] [-n THREADS]

All input workloads are compressed, to know how to uncompress one, type :
		docker run [repository/tag] uncompress-help
